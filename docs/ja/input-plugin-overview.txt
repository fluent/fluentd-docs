# インプットプラグイン概要

Fluentdには３種類のプラグライン(インプット, アウトプットおよびバッファー)があります。本記事ではインプットプラグインの概要を説明します。

## 概要

インプットプラグインはFluentdを拡張し、外部資源からイベントログを検索および取得します。インプットプラグインでは典型的にはスレッドソケットおよびリッスンソケットを作成します。 定期的にデータ資源から、データを取得し書き込むこともできます。

* [in_http](in_http)
* [in_tail](in_tail)
